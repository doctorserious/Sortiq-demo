# Sortiq-demo

SortIQ
Waste sorting education app for PSOAS student housing in Oulu, Finland.

Overview
SortIQ is an interactive mobile web app that helps students learn proper waste sorting through gamified experiences and an AI-powered scanner. Built mobile-first (430px max-width), it provides practical guidance for everyday recycling decisions.
